# Portfolio (Mutarambirwa Joseph)

This repository contains a professional portfolio website in the `site/` directory.

## Contents
- `site/portfolio.html` — main portfolio landing page
- `site/resume.html` — resume and experience page
- `site/projects.html` — project case studies and technical deep dives
- `site/skills.html` — engineering skills and technology matrix

## Deployment
1. Set GitHub Pages source to `main` branch at `/site` or root.
2. Use a custom domain (optional): e.g. `mutarambirwaj.io`.
3. Add CI for accessibility/tests: `npm test` + `pa11y`.
